***
acm
***



===========
Description
===========

 

This is the AWS Certificate Manager (ACM) command line reference. It provides descriptions, syntax, and usage examples for each ACM command. You can use AWS Certificate Manager to request SSL/TLS certificates for your AWS-based websites and applications. For general information about using ACM and for more information about using the console, see the `AWS Certificate Manager User Guide`_ . For more information about using the ACM API, see the `AWS Certificate Manager API Reference`_ .
